About the fields
Username

Your ParentPay user name.

Password

The password for your ParentPay user name.

Import after

The date from which you want to import transactions from ParentPay to Fund Manager. By default the date is the period beginning date.

Note: Authentication is done for you before you complete training. Entry of the ParentPay Username and Password ensures that ParentPay can communicate with Fund Manager.

Use this screen to enter ParentPay credentials and specify an Import after date, if this is different from the default period beginning date.
