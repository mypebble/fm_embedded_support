Bank History

About the fields
Bank Account

The bank account you want to view transactions for.

Type

The type of transaction which may be: All, Cheques, Direct Transactions, Paying-in Slips, Transfers or ePayments.

Status

The status of the transaction which may be: Reconciled or Unreconciled.

Date From

The date you want to start viewing transactions.

Date To

The date you want to stop viewing transactions.

Search

Click Search to display a filtered list of transactions.

List of bank transactions

By default, all bank transactions are listed, until you filter the list displayed using the Bank Account, Type, Status and Date fields.

This report shows the debits and credits associated with the bank account or accounts displayed. It is used during preparation for period end in viewing your unreconciled transactions and it is useful if you need to reprint paying-in slips.

How do I ...

Use the search criteria?
  1. Enter the criteria you want to display the results you want.
  2. Click Search.
Print a paying-in slip?
  • Click on the paying-in slip Reference hyperlink to display the Bank Paying-in Slip and click Print from there.
View unreconciled transactions?
  1. Select the bank account.
  2. Select Unreconciled from the Status drop-down.
  3. Select the period start and end dates in the Date From and Date To boxes.
Print the list of bank transactions?
  • Click Print. A prompt tells you your report is being prepared. Once it has been prepared it is displayed as a PDF and you can then print the PDF file.
Export the list of bank transactions?
  • Click Export to CSV. The list of bank transactions is automatically saved as a download CSVThe name given to a file which stores values that can be read by computer programs. Each line in the file is a record and each record is separated by a comma. file.
