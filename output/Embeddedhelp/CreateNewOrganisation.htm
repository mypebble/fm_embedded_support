How do I ...

Add a new organisation?
  1. Enter the organisation Name and select In Use to ensure you can select the organisation on the system.
  2. Enter the contact details for each contact in the organisation - Name, Address and Post Code. You can add additional contacts by clicking the plus icon alongside the Post Code.
  3. Enter a Phone Number for each contact, using the plus icon alongside to add Phone Number fields as required.

    4. Note: If you want to be able to send SMS text messages to the contact, you must enter the phone number here, and then select it for the pupil, using Administration > Manage Names > Manage Phone Numbers.

  4. Enter an Email Address for each contact, using the plus icon alongside to add Email Address fields as required.
  5. Click Save.
Amend organisation details?
  • Use Administration > Manage Names, select the organisation name to display the Update Organisation screen.
Remove organisation details?
  • Use Administration > Manage Names, select the organisation name to display the Update Organisation screen.
  • Clear the In Use check box and click Save.
