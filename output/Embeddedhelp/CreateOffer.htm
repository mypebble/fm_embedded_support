sQuid Offer Wizard

About the fields
Account

The user account you want to record money into.

Settle With Third Party

Selected means that a third party will pay the amount, rather than the parent.

Title

This is the offer title which parents see on their sQuid portal when looking at the offers.

Optional description

You can add text which describes the offer in more detail.

Amount

The amount of the offer e.g. the amount that must be paid.

Unspecified

Selected means that the amount can be paid by instalments, leaving it up to the parent to decide what to pay towards the amount each time, until all the amount is paid.

Note: Parents will be able to see the full amount on sQuid. Once they start paying towards the trip, the value will reduce until they reach 0 (zero).

Start Date

The date the offer starts and will appear as visible for people to take up.

End Date

The date the offer expires and will be removed.

Deadline

The date by which the school needs payment for the offer.

Min

The minimum amount of slots, items or places available.

Maximum

The maximum number of slots, items or places the parent can buy.

Items Available

The number of slots e.g. seats available on a coach trip.

Recurring

Selected means that the offer will be given more than once. e.g. school uniform offers, ties, badges.

Mandatory checkbox text

Selected means that a parent has to select a checkbox when taking up the offer. This is similar to a Terms and Conditions checkbox when downloading a computer program to your PC.

Optional checkbox text

Selected means that an optional checkbox is displayed for parents when they take up the offer.

List of Names

A list of names of pupils who qualify for the offer.

Note: At least one pupil name must be selected before you can post an offer.

Filter

Enter a character or characters in the name you want to find and the list of displayed names adjusts accordingly to show a shorter list to choose the name from.

Note: The filter works by displaying all names which contain the character or characters you have entered.

Use this screen to create offers for parents.

How do I ...

Post an offer?
  1. Select an account and if the amount will be paid by a third party, select Settle With Third Party.
  2. Enter the offer Title and a description, if required.
  3. Enter the cost of the offer in the Amount field.
  4. If you want to offer payment by instalments, then select Unspecified.
  5. Enter the Offer Dates.
  6. Enter details about the offer, the Minimum number of items available, the Maximum number each parent can purchase and the maximum number of Items Available.
  7. If you want the offer to be recurring (i.e. you will offer the same thing more than once), select Recurring.
  8. If you want to force parents to select terms and conditions for the offer, enter text in the Mandatory checkbox text field.
  9. If you have optional text you want parents to acknowledge for the offer, enter text in the Optional checkbox text field.
  10. Select the Names of the pupils who qualify for the offer.
  11. Click Post Offer.
Post a recurring offer?
  • Select the Recurring check box.
Offer payment by instalments?
  • Select Unspecified in the Details section.
Ensure parents agree to terms and conditions of the offer?
  • Enter text in the Mandatory checkbox text field.
