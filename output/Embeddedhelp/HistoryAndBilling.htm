Message History

About the fields
Filter

Enter a character or characters in the message you want to find and the list of displayed messages adjusts accordingly to show a shorter list to choose from.

Note: The filter works by displaying all messages which contain the character or characters you have entered.

List of Messages

A list of all messages which you have sent is shown, displaying: message Reference number, Date sent, Status (whether the message was sent or not), Cost of sending the message, the Phone number the message was sent to, the Name of the person it was sent to and the Text that was sent in the message.

Note: A charge of 5p per message is standard.

Use this screen to review the SMS text messages which you have sent.

How do I ...

Find a message?
  • Enter the message Ref you want to find (if you know it) in the Filter box.
  • Use the Filter to shorten the list of messages that is displayed, by entering a character or characters you think are in the message.
Check my billing?
  • Click the Billing tab to view the cost of messages, balance outstanding and create invoices or record payment for the message charges..
