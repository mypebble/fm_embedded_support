About the fields
ePayments File

The CSVThe name given to a file which stores values that can be read by computer programs. Each line in the file is a record and each record is separated by a comma. file which contains the list of transactions that you want to import into Fund Manager.

Upload

Uploads the CSV file to Fund Manager and displays the transactions to import at the foot of the screen.

Account

The Account to allocate the transactions against.

Settlement account

The Settlement account to allocate the transactions against. You must select a settlement account or you won’t be able to import any transactions. The settlement account lets you record individual ePayments in Fund Manager. Then once you have imported you can transfer the total amount from the Settlement account into your desired bank account.

List of transactions

The list of transactions contained in the import file, shows: the transaction Date, transaction Name, Class, Year, Amount and Description.

You must match all transactions to import by selecting a To Name in Fund Manager.

Import files are CSVThe name given to a file which stores values that can be read by computer programs. Each line in the file is a record and each record is separated by a comma. files which are created externally to Fund Manager. For example, within SchoolComms, SchoolMoney or ParentMail.

Note: You can set up ParentPay , sQuid and Joinos so that transactions are automatically imported from those packages. For information on how to do this, see Import ParentPay Transactions Automatically, Using sQuid with Fund Manager and the Joinos User Guide.

How do I ...

Import ePayments from a new file
  1. Select the CSV file to import.
  2. Click Upload.
  3. Select the Account and Settlement Account to allocate the transactions against.

    4. If you don’t have a settlement account to select, create one via Administration > Manage Bank Accounts > Create Bank Account and follow the instructions on the Create New Account screen to create a settlement account.

  4. Work through the list of transactions, selecting and matching To Name from the drop-down list of names

    5. Warning! You should only import one page at a time, so ensure you click Import when you match all the transactions to the To Name on each page.

    The closest matching name appears at the top of the list. If the correct name is not available for you to select, you may need to create a new name, via Administration > Manage Names > Create New Name and follow the instructions on the Create New Pupil or Create New Organisation tab.

  5. Click Import to import the transactions on the displayed page of transactions.

    6. When you successfully import a page of transactions, a View Pending Import button appears alongside the Upload button.

    You can leave the screen and return later. If you do, don’t upload the file again. instead click on View Pending Import and carry on from the point you left the screen.

 
Continue an import which was interrupted

If you have navigated away from the screen after importing some transactions, don’t click Upload File to import the entire file again, instead click View Pending Import and carry on from where you had left off before.

Transfer total amount from settlement account
  • Go to Bank & Cash > Transfer between Bank Accounts and follow the instructions on the screen there.
