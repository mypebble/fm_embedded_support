Pull New sQuid Sales

Pulls transactions from sQuid into Fund Manager.

Use this screen to pull transactions from sQuid into Fund Manager and to import transactions for the pupils and user accounts.

How do I ...

Pull transactions from sQuid into Fund Manager?
  1. Enter the start and end dates
  2. Click Pull New sQuid Sales.
Import transactions?
  1. Select the Settlement Account for the money to be imported into.
  2. Select the transactions to be imported.
  3. Click Import.

    4. Once imported, the transactions show in Fund Manager against the correct user accounts and names. The money is added to the sQuid Settlement Account.
