List of Cancellations

About the fields
Start Report On

Select the start date for the report.

End Report On

Select the end date for the report.

List Transactions

The list of transactions between the start and end date for the report, selected before clicking List Transactions.

The list displays the transaction Date, Amount, Type, Number, Account, Name, Class, DescriptionCancelled On and Reason.

Filter

Enter a character or characters in the transaction you want to find and the list of displayed transactions adjusts accordingly to show a shorter list.

This report lets you see a list of cancelled transactions between two dates. .

How do I ...

Use the date criteria?
  1. Enter the start and end date to display the results you want.
  2. Click List Transactions.
Change the display sequence of the cancellations?
  • Click the hyperlink column name to re-order the display. For example, clicking on Date will order the display in ascending or descending order. Clicking on Account. will order alphabetically. Clicking on Amount will order by amount.
