Manage Gift Donors

About the fields
List of Names

A list of names is displayed with the following information for each name: Class, Year, Registration number and whether the name has a gift aid donor associated with it or not.

Filter

Enter a character or characters in the name you want to find and the list of displayed names adjusts accordingly to show a shorter list to choose the name from.

Note: The filter works by displaying all names which contain the character or characters you have entered.

In order for eligible transactions to be included in a GiftAid claim, a designated GiftAid donor must be selected for the pupil. Use this screen to select the donor associated with each name.

What other settings are needed for GiftAid donation?
  • The Gift Aid checkbox must be selected when you enter or amend a receipt.
  • The Gift Aid donor checkbox on the Update Pupil screen must be selected.

How do I ...

Change the display sequence of the names?
  • Click the hyperlink column name to re-order the display. For example, clicking on Name will order the display in ascending or descending order. Clicking on Class. will order by class.
Mark a pupil as eligible for gift aid?
  • Click the hyperlinked name to display the Update Pupil screen and select the Gift Aid donor checkbox.
Select a designated GiftAid donor name for the pupil?
  • Select a name from the GiftAid Donor drop-down list.
