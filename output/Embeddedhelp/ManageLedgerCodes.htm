Manage Ledger Codes

About the fields
Code

A code which is used to group transactions for a particular purpose.

The code can be alphanumericNumbers, letters or a mix of both., up to 20 characters. If you integrate with an external accounting package for reporting purposes, set up the codes in Fund Manager that match the same codes in the external system you are integrating with. This ensures that the transaction appears in the external ledger that you want it to, making it easy for you to identify and track the transactions.

Bank Ledger Code

A code which is used to identify banks for reporting purposes.

The code can be alphanumericNumbers, letters or a mix of both., up to 20 characters. If you integrate with an external accounting package for reporting purposes, set up the codes in Fund Manager that match the same codes in the external system you are integrating with. This ensures that the transaction appears in the external ledger that you want it to, making it easy for you to identify and track the transactions.

Note: You can only set up Bank Ledger Codes for Bank accounts that have already been set up using Administration > Manage Bank Accounts.

Description

The name of the code.

Note: The VAT and Cash in Hand Ledger Codes are set up by default and their Description cannot be changed. However, you can enter a Code name for these default codes so that they match the code names in your external financial system.
