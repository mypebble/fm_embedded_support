Name Summary

About the fields
Account

The user account taken items have been recorded against. For example, Dinners, Music Club, Family Burns Night Ceilidh.

Group

The group of pupils who qualify for an item taken.

Date From/To

The start date of the week the items were taken.

Show Paid Balances

Click Search to display a list of all pupils in the Group and Class selected. alongside a grid for the week.

Summary

The unit cost for the item, which has been defined using Administration > Update Account.

List of accounts

A list of pupils (who are in the selected group and class) is displayed along with the items taken during the week.

Filter

Enter a character or characters in the name you want to find and the list of displayed names adjusts accordingly to show a shorter list to choose the name from.

Note: The filter works by displaying all names which contain the character or characters you have entered.

Use this screen to see individual pupil balances.

How do I ...

Find pupils?
  • Enter the Name you want to find (if you know it) in the Filter box.
  • Use the Filter to shorten the list of names that is displayed, by entering a character or characters you think are in the name.
  • If you aren’t sure if the name is out of use, you can show or hide out of use names in the following way:
    • Show out of use names?
      • Click Show out of use.

      Out of use names are shown greyed out in the List of Names

    • Hide out of use names?
      • Click Hide out of use.

      Only in use names are shown in the List of Names.

  • Use the paging controls to check for the name you want on another page.
See the balance of a pupil’s account?
  1. Select the Account.
  2. Select the Group, if applicable.
  3. Select the Date From/To dates.
  4. To see all balances, including paid balances, select Show Paid Balances. If you only want to see outstanding balances, clear the Show Paid Balances check box.
  5. Click Search.

    6. The Receipts, Payments, Balance and Final Balance columns show the amount (£) for each pupil. The Paid, Free and Total columns are a count of the meals taken. The Free column next to Class indicates whether the pupil is eligible for Free School Meals.

Print the balances?
  • Click Print.
Export balances?
  • Export a list of balances to a CSVThe name given to a file which stores values that can be read by computer programs. Each line in the file is a record and each record is separated by a comma. file by clicking Export CSV. This list is downloaded as name_summary.csv.
View what has been taken in different ways?
  • To see what items have been taken, use Register > Taken Items.
  • To see what items a pupil has been recorded against, use Register > Statement.
  • To see the total number of free meals and universal free school meals, use Register > Universal Free School Meals.
