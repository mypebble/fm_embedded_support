About the fields
List of Names

A list of names, showing transaction: Name, Class, Year, Reg and Groups.

Filter

Enter a character or characters in the name you want to find and the list of displayed names adjusts accordingly to show a shorter list to choose the name from.

Note: The filter works by displaying all names which contain the character or characters you have entered.

Warning! If you are an IMPACT school and receive your card numbers from a catering company, you must talk to us about your file before attempting to register those pupils.

How do I ...

Find Names?
  • Enter the Name you want to find (if you know it) in the Filter box.
  • Use the Filter to shorten the list of names that is displayed, by entering a character or characters you think are in the name.
Register pupil names?
  • Select the pupil names you want to register and click Register Names.

    • Tip: Use the Filter box to shorten the list to find the names you want and then select the Select all filtered checkbox.

    Information is automatically sent to sQuid and when sQuid completes, you can see the names, using the View Registerd Names tab.
