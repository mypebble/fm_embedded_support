Upload Taken File

About the fields
Taken file

A CSVThe name given to a file which stores values that can be read by computer programs. Each line in the file is a record and each record is separated by a comma. file which contains details of items taken and has been generated outside Fund Manager. It can be uploaded by selecting the file and clicking Upload.

Event date

The date the file was generated outside Fund Manager.

Cost Centre

The user account the items taken are to be uploaded and associated with.

Overwrite

Selected means that the uploaded file will overwrite the existing items taken for the account in Fund Manager. To append the items taken during the upload to items already existing in Fund Manager, clear the Overwrite check box.

Use this screen to upload a CSV taken file for a school, where the automatic process has failed for some reason.

How do I ...

Upload a CSV file of taken items?
  1. Select the Taken file.
  2. Select the Event date - this is the date when the CSV file was automatically generated outside Fund Manager, due to a system failure of some kind.
  3. Select the user account from the Cost centre drop-down. This is the user account you want to record the uploaded taken items against.
  4. If you want to overwrite all items existing in Fund Manager against the account then select Overwrite. If you want to append the items taken in the upload file to any that already exist in Fund Manager, clear the Overwrite check box.
  5. Click Upload.
