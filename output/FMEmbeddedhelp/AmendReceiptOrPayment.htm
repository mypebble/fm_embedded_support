About the fields
Filter

Enter a character or characters of the transaction you want to find and the list of displayed transactions adjusts accordingly to show a shorter list to choose the transaction from.

List of transactions

A list of transactions is displayed with the following information for each transaction: transaction date, user name, transaction description, type (e.g. Cash, Cheque), transaction reference and receipt amount or payment amount.

How do I ...

Amend a receipt or payment?
  1. Select the transaction you want to amend.

    2. Tip: Use the Filter to find the transaction. Enter a character or characters in what you want to find to show a shorter list to select from.

  2. Click the Description of the transaction you want to amend and follow the instructions on the Amend Receipt or Amend Payment screen.
Set the VAT Status for receipts or payments?

Only applicable to transactions in the current period which have not been paid into the bank, reconciled or included in the VAT 100 report.

  • Select Set VAT Status and follow the instructions on the Set VAT Status screen.
set the Ledger Codes for receipts or payments?

Only applicable to transactions in the current period which have not been paid into the bank, reconciled or included in the VAT 100 report.

  • Select Set Ledger Codes and follow the instructions on the Set Ledger Codes screen.
