Cancel/Amend Bank Transaction

List of transactions

A list of transactions that have not been reconciled is displayed. For each transaction, the Date, Code, Reference Number, Details, Debit or Credit values are shown.

Use this screen to cancel or amend bank transactions that have not been reconciled.

How do I ...

Amend a paying-in slip?
  • Click the hyperlinked paying-in slip Reference Number and follow the instructions on the Amend Paying-in Slip screen.
Cancel a transaction?
  • Select the transaction you want to cancel and click Cancel.

    • The selected transaction is displayed so you can check you have the one you want to cancel.

  • If this is the transaction you want to cancel, click Cancel Transaction.
