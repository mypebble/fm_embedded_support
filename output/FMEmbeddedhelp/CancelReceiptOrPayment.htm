Cancel Receipt or Payment Transaction

About the fields
Filter

Enter a character or characters of the transaction you want to find and the list of displayed transactions adjusts accordingly to show a shorter list to choose the transaction from.

List of transactions

A list of transactions is displayed with the following information for each transaction: transaction date, user account code, user name, user’s class name, transaction description, type (e.g. Cash Receipt, Cheque Payment), amount, transaction number (e.g. Receipt Number or Chq/Vouch No) and the transaction reference.

About cancelling transactions

If a transaction has been included in a VAT 100 Report, or has been left unreconciled from a previous closed period, you cannot cancel it. However, you can enter a contra-entry transaction which has the effect of cancelling the original transaction, using Transactions > Enter a Receipt or Transactions > Record a Payment.

How do I ...

Cancel a receipt or payment
  1. Select the transaction you want to cancel.

    2. Tip: Use the Filter to find the transaction. Enter transaction character detail to show a shorter list to select from.

  2. Select the receipt or payment you want to cancel.

    3. Tip: You can select more than one transaction to cancel but you will only be able to give one reason for the cancellation on the Confirm Cancellation screen. If you have different reasons for cancelling different transactions, and you want to record the reason for each that you cancel, select and cancel each transaction individually.

    The Amount and the Number of what you select is displayed in the box at the right of the screen.

  3. Click Continue.
  4. On the Confirm Cancellation screen which appears:
    5. About the fields on the Confirm Cancellation screen
    Selected Receipts

    Enter a character or characters of the transaction you want to find and the list of displayed transactions adjusts accordingly to show a shorter list to choose the transaction from.

    Selected Payments

    A list of transactions is displayed with the following information for each transaction: transaction date, user account code, user name, user’s class name, transaction description, type (e.g. Cash Receipt, Cheque Payment), amount, transaction number (e.g. Receipt Number or Chq/Vouch No) and the transaction reference.

    Reason for Cancellation

    The reason why you are cancelling the transaction.

    1. Review the transactions you have selected to ensure you are cancelling the transactions you want to cancel.
    2. Enter a reason for the cancellation.

      3. Note: You can cancel many transactions at once but you can only give one reason for the cancellation. If you want to give separate reasons for cancelling individual transactions you must select and confirm each transaction separately.

    3. Click Cancel Transaction.
Cancel or amend a paying-in slip

If you have made a mistake on a paying-in slip and you want to amend or cancel this, use the Bank & Cash > Cancel/Amend a Bank Transaction option.

Cancel items from a previous year

If you have uncashed cheques and receipts left over from a closed period and you want to clear these from the reconciliation page and retain a trail for audit purposes, you can enter contra-entry transactions, using Transactions > Enter a Receipt or Transactions > Record a Payment.

Once you save those transactions, you will have your the original transactions on your bank reconciliation screen along with the matching contra-entriy transactions. You can then reconcile these together and they will be cleared from the reconciliation screen.

Cancel transactions that have been included in the VAT 100 Report

You cannot cancel a VAT transaction that has been included in the VAT 100 report.
