Compose Joinos Message

About the fields
Title

The title of the message.

Body

The body of the message.

Attachments

Attachments which can be selected to attach to the message in Joinos e.g. Newsletters. Attachments must be uploaded first, using Offers > Upload Document.

Note: Attachments are attached to the Joinos message within Joinos. They are not attached to the emails that are sent out.

List of contacts

The list of contacts you can send a message or an sms message to. You can only send email messages if the contact has an email entered. And you can only send sms text messages to contacts if they have a Primary Phone selected.

Tip: You can check this by clicking Manage Names.

Filter

Enter a character or characters in the name you want to find and the list of displayed names adjusts accordingly to show a shorter list to choose the name from.

Note: The filter works by displaying all names which contain the character or characters you have entered.

Use this screen to compose and send email messages and sms text messages to parents and carers of pupils.

Note: Email addresses and phone numbers are added to contacts through Administration > Manage Names. Attachments to send are uploaded using Offers > Upload Document.

How do I ...

Send an email message?
  1. Enter the Title and the Body of the message.
  2. Select the names in the Contact list to send the message to.
  3. Click Send.
Send an attachment with a message?
  • Select any of the listed attachments to attach to the message in Joinos.

Note: Attachments are not attached to the email message but to the message within Joinos. The attachments are accesssed within parents Joinos accounts.

Send an SMS alert?
  • Select the check box Send selected contacts a message alert via SMS.
