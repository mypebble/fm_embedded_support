About the fields
Issuer ID

This is always sQuid.

SCH Password

The school password, provided to Pebble by sQuid.

OMS Username

The OMS username provided to Pebble by sQuid.

OMS Password

The OMS password provided to Pebble by sQuid.

SCH URL

The school URL - either Production, Pre-Prod or IP Address. This information is provided to Pebble by sQuid.

OMS URL

The OMS URL - either Production, Pre-Prod or Staging. This information is provided to Pebble by sQuid.
