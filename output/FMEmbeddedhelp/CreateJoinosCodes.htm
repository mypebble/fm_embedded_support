About the fields
List of Names

The list of pupil names is displayed, showing: Name, Class, Year, Reg, Groups, GA (gift aid) and J (joinos).

Show Registered/Hide Registered

Click Show Registered to show those who are registered for Joinos.

Click Hide Registered to hide those who are registered for Joinos.

Use this screen to select students and staff who are waiting to be registered on Joinos and create Joinos codes for them. The list includes all new students and staff at the school or those who have never been registered.

Joinos codes are used by parents to access Joinos and pay for trips and activities for their child.

Note: Joinos codes expire after three months so if a pupil has already registered but their parent or carer has never used the code you gave them, you will need to recreate a Joinos code for them.

How do I ...

Create Joinos codes?
  1. Select the pupils to generate Joinos codes for.
  2. Click Create Joinos Codes.

    3. Joinos codes are created and displayed on the Joinos Codes tab.

Recreate a code for a pupil whose code has expired?
  1. Click Show Registered to show the pupil’s name in the list of pupils.
  2. Select the pupil’s name and click Create Joinos Codes.

    3. The pupil’s new code is displayed on the Joinos Codes tab.
