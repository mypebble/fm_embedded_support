Create New Pupil

Note: Most names are added, updated or amended via a pupil or staff import of names through MISapp or a different MIS system.

About the fields
Forename

The pupil’s first name.

Surname

The pupil’s surname.

Upn

The unique pupil number allocated to new pupils in MISapp and imported with other pupil data from MISapp.

Gift Aid Donor

Selected indicates that the pupil is associated with a Gift Aid Donor name and so donations recorded for the pupil qualify for gift aid.

In Use

Selected indicates that the pupil record is in use.

Year

The year the pupil is in.

Class

The class the pupil is in.

Reg Group

The registration group the pupil is in.

Contact Name

A contact name for the pupil, for example, a parent who is in charge of providing the pupil with money for school activities.

Note: You can enter details for multiple contact details by clicking the plus icon to the right of the Post Code field.

Address

A contact address for the pupil.

Post Code

The post code for the contact address.

Phone Number

The phone number for the contact.

You can enter multiple phone numbers if you have these (for example, mobile, landline, work, different numbers for multiple contacts etc). To enter an additional telephone number, click the plus icon to the right of the Phone Number field

Note: If you want to send SMS text messages to the contact, the phone number must be entered here.

Email Address

The email address for the contact.

You can enter multiple email addresses if you have these (for example, work and home email addresses for a contact, or email addresses for multiple contacts. To enter an additional email address, click the plus icon to the right of the Email Address field

You can import pupil details from MISapp or using another information management tool. But you can also create a pupil’s details manually within Fund Manager. Use this screen to record pupil details manually.

How do I ...

Add a new pupil?

Note: Most names are added, updated or amended via a pupil or staff import of names through MISapp or a different MIS system.

  1. Enter the pupil Forename, Surname, Upn (if required), and select In Use to ensure you can select the pupil on the system.
  2. If you know that their contact can GIft Aid donations, select the Gift Aid Donor box.
  3. Select the Year, Class and Registration Group for the pupil from the drop-down lists.

    4. Note: Years, classes and registration groups are set up as groups, using Administration >  Manage Names.

  4. Enter the contact details for each contact - Name, Address and Post Code. You can add additional contacts by clicking the plus icon alongside the Post Code.
  5. Enter a Phone Number for each contact, using the plus icon alongside to add Phone Number fields as required.

    6. Note: If you want to be able to send SMS text messages to the contact, you must enter the phone number here, and then select it for the pupil, using Administration > Manage Names > Manage Phone Numbers.

  6. Enter an Email Address for each contact, using the plus icon alongside to add Email Address fields as required.
  7. Click Save.
Amend pupil details?

Note: Most names are added, updated or amended via a pupil or staff import of names through MISapp or a different MIS system.

  • Use Administration > Manage Names, select the pupil name to display the Update Pupil screen.
Remove pupil details?
  • Use Administration > Manage Names, select the pupil name to display the Update Pupil screen.
  • Clear the In Use check box and click Save.
