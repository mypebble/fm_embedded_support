Joinos Message History

About the fields
List of Sent Messages

A list of messages that have been sent is displayed, showing: date Sent, message Title and Body.

Filter

Enter a character or characters you want to find and the list of displayed messages adjusts accordingly to show a shorter list to choose from.

Note: The filter works by displaying all messages which contain the character or characters you have entered.

Use this screen to review the messages that have been sent.

How do I ...

See the detail of a sent message?
  • Click the hyperlinked message Title to display the Sent Joinos Message screen.
    • What can I see on the Sent Joinos Message screen?

    The Title and Body of the message are shown at the top of the screen. At the bottom of the screen contacts who have received the message are listed, showing: the contact Name, Class, Year, Registration group, Groups and whether the message has been Read or not.

    You can filter the list if it is long to check if a particular contact has received the message and you can sort the list of contacts by clicking the column title, Name to order the display alphabetically.

Check if a recipient has read my message?
  • Click the hyperlinked message Title to display the Sent Joinos Message screen.

    • At the bottom of the screen, use the Filter to find the recipient and check the Read column to see if they have read the message or not.
