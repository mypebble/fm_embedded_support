Journal Report

About the fields
Date From/To

Select the start and end date for the report.

Ledger Code

Select this to show journals in the ledger.

Bank/Cash

Select this to show journals in the bank/cash accounts.

Search

Click to search using your criteria. The list of journals found is displayed as a result of the search.

List of Journals

The list of journals between the start and end date for the report, in the ledger and bank or cash account, selected before clicking Search.
