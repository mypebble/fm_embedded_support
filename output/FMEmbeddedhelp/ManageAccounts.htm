Manage Accounts

About the fields
List of Accounts

A list of accounts that have been created, is displayed. The list displays the following information for each account: Code, Account, Groups, Budget, Unit Cost, Balance and the account status (S) - whether the account is in use or not.

From here you can create a new name or organisation by clicking Create New Account.

Filter

Enter a character or characters in the account you want to find and the list of displayed accounts adjusts accordingly to show a shorter list to choose from.

Note: The filter works by displaying all accounts which contain the character or characters you have entered.

Show out of use / Hide out of use

Show out of use will include all accounts marked as Out of use in the list of accounts displayed. If an account is out of use, it is greyed out in the list so that you can easily recognise that it is not in use.

Hide out of use only shows accounts in the list which are marked as In use.

Batch Actions

You can select batches of accounts in the list and mark them as In Use or Out of Use with one click of Update Name(s).

You can make it easy to apply batch actions to names by creating groups. See Group Actions.

Group Actions

You can create groups to assist you in selecting accounts quickly when you want to find transactions for the accounts in that group, or include grouped accounts in the reports.

Groups are easily managed from the Group Actions section of the Manage Accounts screen.

Note: Accounts can be assigned to more than one group, so you can design the groups that will make your workday easier.

Add and manage accounts for events, classes or locations against which you can record incoming and outgoing transactions. These accounts are not bank accounts. They represent individual column headers in a manual ledger. Recording transactions against these accounts allows you to track and report on the money flow for the different events, classes or locations.

How do I ...

Create accounts?
  • Click Create New Account and follow the instructions on the Create New Account screen..
Find accounts?
  • Enter the Account you want to find (if you know it) in the Filter box.
  • Use the Filter to shorten the list of names that is displayed, by entering a character or characters you think are in the account.
  • If you aren’t sure if the account is out of use, you can show or hide out of use accounts in the following way:
    • Show out of use accounts?
      • Click Show out of use.

      Out of use accounts are shown greyed out in the List of Accounts

    • Hide out of use accounts?
      • Click Hide out of use

      Only in use accounts are shown in the List of Accounts.

Edit account details
  • Click the hyperlinked Account and follow the instructions on the Update Account screen.
Manage Groups?

Groups assist you in selecting accounts quickly when you want to assign an action to the accounts in the group.

Groups are easily managed from the Group Actions section of the Manage Accounts screen.

Note: Accounts can be assigned to more than one group, so you can design the groups that will make your workday easier.

Create Group

Add a new group by clicking Create Group. The Manage Groups screen appears.

  1. Enter a name for the group.
  2. Click Add Group. Once you have done this you will be able to select it from the Select Group drop-down list.
Select Group

Select a group you have already created from the drop-down list of groups available.

Add to Group
  1. Select the accounts in the List of Accounts that you want to add to a group.
  2. Select the group you want to add the accounts to from the Select Group drop-down list.
  3. Click Add to Group.

    4. The group name is added to the Groups column for the selected accounts.

Remove from Group
  1. Select the accounts in the List of Accounts that you want to remove from the group.
  2. Select the group from the Select Group drop-down list.
  3. Click Remove from Group.

    4. The group name is removed from the Groups column for the selected accounts .

Delete Group

Note: You can only delete groups which don’t have any accounts assigned.

  1. Select the group you want to delete from the Select Group drop-down list.
  2. Click Delete Group.
  3. At the prompt to confirm that you really want to delete the group, click Delete.
Use Batch Actions?

The batch actions available are to mark accounts In Use or Out of Use.

  1. Select the accounts in the List of Accounts that you want to apply the batch actions to.
    • If you have created a Group for these accounts, you can select the Group name.
    • If you want to select out of use items, ensure that you click Show out of use to display these accounts.
  2. Select In Use to mark all the selected accounts as in use, or Out of Use to mark all the selected accounts as out of use.
  3. Click Update Account(s) to apply the action to the accounts.

    4. Note: When you have applied the action, clear the selection from the account before applying further changes. It isn’t removed automatically.
