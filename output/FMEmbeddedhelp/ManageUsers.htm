Manage Users

About the fields
List of Names

The list of user names that have been created, is displayed. The list displays the following information for each name: Name, Email, Last Login and user Status - whether the user is in use or not.

Filter

Enter a character or characters in the name you want to find and the list of displayed names adjusts accordingly to show a shorter list to choose the name from.

Note: The filter works by displaying all names which contain the character or characters you have entered.

Batch Actions

You can select batches of names in the list and mark them as In Use or Out of Use with one click of Update User(s).

You can make it easy to apply batch actions to names by creating groups. See Group Actions.

Use this screen to add and manage Fund Manager users.

How do I ...

Find Users?
  • Enter the Name you want to find (if you know it) in the Filter box.
  • Use the Filter to shorten the list of users that is displayed, by entering a character or characters you think are in the user name.
Create Users?
  • Click Create New User and follow the instructions on the Create New User screen.
Update Users?
  • Click the hyperlinked user Name to display the Update User screen.
Use Batch Actions?

The batch actions available are to mark names In Use or Out of Use.

  1. Select the users in the List of Names that you want to apply the batch actions to.
  2. Select In Use to mark all the selected names as in use, or Out of Use to mark all the selected names as out of use.
  3. Click Update Names to apply the action to the names.

    4. Note: When you have applied the action, clear the selection from the name before applying further changes. It isn’t removed automatically.
