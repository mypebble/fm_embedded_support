About the fields
Primary Record

The master pupil record which is used for the consolidation. It shows the pupil Name, Class, Year, Reg and Groups.

Secondary Record

The secondary pupil record which is being consolidated into the Primary Record. It shows the pupil Name, Class, Year, Reg and Groups.

Consolidated Record

The pupil record as it will be consolidated, showing Name, Class Year, Reg and Groups.

Affected by this consolidation

A list of items which are affected by the consolidation. The list shows the Type of item, e.g. Receipt, Payment, etc, the Number Being Moved, the Total Value Transferred and the Number Being Discarded.
