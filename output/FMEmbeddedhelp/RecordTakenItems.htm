Record Taken Items

About the fields
Account

The user account that you record the taken items against. For example, Dinners, Music Club, Family Burns Night Ceilidh.

Note: The account is created using Administration > Manage Accounts and you must select the check box Register Account when you create it in order to see it appear in the drop-down list of user accounts.

Group

The pupils you have created a group for to assist in your search for particular pupils..

Note: The group is created through Administration > Manage Names > Create Group.

Class

The class you want to record attendance for.

W/C

The start date of the week you want to record attendance for an item.

Search

Click Search to display a list of all pupils in the Group and Class selected. alongside a grid for the week.

List of pupils

The list of pupils you can record attendance against.

Filter

Enter a character or characters in the name you want to find and the list of displayed names adjusts accordingly to show a shorter list to choose the name from.

Note: The filter works by displaying all names which contain the character or characters you have entered.

Use this screen to record attendance or items taken by pupils, for example, meals, during a week.

Note: You must have a Register Account set up to record things against. For example, an account for Attendance, an account for Meals, and so on. Money is received for a Register Account through Fund Manager and then you record the attendance or meals taken by individual pupils here.

How do I ...

Record items taken?
  1. Select the Account.
  2. Select the Group and / or Class, if applicable.
    3. Note: For example, if you have a group set up for an after school club and a number of pupils assigned to the club, you would select this group to display all pupils who attend the after school club. You would not need to select a class as pupils from a number of classes attend the club. When you click Search, a list of all pupils in the after school club is displayed and you can then record which pupils were present at the club on which days of the week.
  3. Select the week commencing (W/C) date.
  4. Click Search.
  5. Select the check box next to each pupil you want to record an entry for on each day that is applicable in the week.
  6. If the amount for the pupil is different from the unit cost on the Accounts screen, enter the difference in the amount box that appears alongside the check box when you select it. This is usual if an amount differs from the Unit Cost of the item for any reason. In that case, enter the amount that differs. The pupil will be charged this amount.

Note: Your entries are automatically saved. Any amendments you need to make are also automatically saved.

Record Universal Infant Free School Meals taken?
What you need to have done first
  • Create a UIFSM group (use Administration > Manage Names).
  • Assign pupils to the group (use Administration > Manage Names).
  • Assign the UIFSM group to the Register account. (use Administration > Manage Accounts).

For more information, see How do I record the Universal Infant Free School Meals being taken?

In Register > Record Items Taken:

  1. Select the Account.
  2. Select the UIFSM Group.
  3. Select the week commencing (W/C) date.
  4. Click Search.
  5. Select the check box next to each pupil you want to record an entry for on each day that is applicable in the week.
