Search Account Balances

About the fields
Account

The name of the account you want to search for.

You can enter a character or characters in the account you want to find in the Search box and choose from a shorter selection of accounts or click the arrow on the Select Account box to select the account from the drop-down list.

Note: You can search by Account or Account Group, not both.

Account Group

A group the account belongs to. Accounts can be grouped to make it easier to find or report on the account. Creation and adding accounts to groups are done through Administration > Manage Accounts.

Note: You can search by Account or Account Group, not both.

Date From/To

Search for transactions which fall within a period of time, determined by the Date From and Date To selected dates.

Include VAT

Selected will include a breakdown of Net Balance, VAT and Gross Balance in the list of accounts.

Sort into Account Groups

Selected will divide the display by account groups.

Only Changed Balances

Selected will display only the accounts whose balance has changed.

Include Transfers

Selected includes accounts where transfers have been made in or out of the account funds.

Include Out of Use

Selected includes accounts which have been marked as out of use.

Search

Click Search to search the list of accounts, using the criteria you have entered and to display the results in the List of Accounts.

Account Totals

This shows the total number of Accounts, the Movement in Period amount, Opening Balances, Receipts amount, Payments amount and Closing Balance.

If you select Include VAT, this also shows a balance breakdown in Net Balance, VAT and Gross Balance.

If you select Sort into Account Groups, the Subtotal for each group is listed at the foot of the group list.

If you select Include Transfers, the list includes Transfers In and Transfers Out.

List of Accounts

The list of accounts, displayed as a result of the search criteria entered before clicking Search.

If no criterion is entered, the list displays the account Code, Account name, Status (displays if the account is in use), Opening balance, Receipts amount, Payments amount and Balance.

If you select Include VAT, the list includes balance breakdown into Net Balance, VAT and Gross Balance.

If you select Include Transfers, the list includes Transfers In and Transfers Out.

This report is comprehensive and allows you the greatest flexibility in searching your accounts. It is often used to determine which accounts are to be closed and which are to be retained with their balance into the next financial period.

Note: If you have outstanding balances in out-of-use accounts, we recommend that you transfer these balances to an account which is in use, preferably a year end closing account, so it will be easier for you to identify this pot of money. If you donâ€™t want to do this then you must select Include Out of Use on the Account Balances report so that your overall year end figures will show your true balance for the year. You should also select Include Transfers in the Account Balances report to show internal movement of funds.

How do I ...

Use the search criteria?
  1. Enter the criteria you want to the results to display. For example, to see transactions between a period of dates select the Date From/To dates, or to see account transfers, select Include Transfers.
  2. Click Search.

Note: You can search by Account or Account Group, not both.

Print accounts list?
  • Click Print. A prompt tells you your report is being prepared. Once it has been prepared it is displayed as a PDF and you can then print the PDF file.
Export accounts list?
  • Export a list of the accounts to a CSVThe name given to a file which stores values that can be read by computer programs. Each line in the file is a record and each record is separated by a comma. file by clicking Export CSV. The list of accounts is automatically exported to a report.csv file.
