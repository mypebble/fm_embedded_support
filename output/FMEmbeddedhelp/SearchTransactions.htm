Search Transactions

About the fields
Account

The name of the account you want to search for.

Note: You can search for more than one account by clicking the plus icon .

Group

A group the name on the transaction belongs to. Names can be grouped to make it easier to find or report on the transaction. Creation of groups and adding names to groups are done through Administration > Manage Names.

Type

Search for a type of transaction e.g. Direct Credit or ePayment.

Name

Select a name on the transaction.

Class

Select a class to display transactions for a class name.

Year

Select a year to display the transactions for a class year.

Date From/To

Search for transactions which fall within a period of time, determined by the Date From and Date To selected dates.

Amount

Enter an amount to display transactions with the amount. You can select to display amounts from All Transactions, Gift Aid Only transactions or VAT Only transactions.

Keyword

Enter a keyword to search for in the transaction description, for example, donation from x, payment from x, payment to x etc.

Aggregate

Select By Name if you want to view transactions for a pupil, aggregated in one Receipt, Payment and Balance figure.

Search

Click Search to search the transactions, using the criteria you have entered and to display the results in the List of Transactions.

Transactions Totals

This shows the total number of Transactions, Accounts and Names and the total amount of Transfers In, Transfers Out, Receipts and Payments and the Total balance for all transactions.

If you select Include VAT, this also shows a balance breakdown in Net Balance, VAT and Gross Balance.

List of Transactions

The list of transactions, displayed as a result of the search criteria entered before clicking Search.

The list displays the transaction Date, user Account, pupil Name, pupil Class, transaction Description + Notes, transaction Type, Receipt amount, Payment amount , transaction Reference and the GA (gift aid) status of the transaction.

This report is particularly useful for finding and viewing different types of transactions, for example, transactions which have VAT included or those which you have imported using Transactions > Import ePayments.

If you select a single account, the Search Transactions report will display the same values and balances as the Accounts Summary report, however here the transfer values are included by default. On the Search Transactions report transfer values are specifically identified as transfers and are not added to the receipts and payments totals.

How do I ...

Use the search criteria?
  1. Enter the criteria you want to display the results you want.
  2. Click Search.
Search for transactions on more than one account?
  • Use the plus icon to add - Select Account - fields as required.
Change the display of the transactions?
  • Click the hyperlink column name to re-order the display of the transactions. For example, clicking on Date will order the display by ascending or descending order. Clicking on Type. will order alphabetically by type.
Print transactions list?
  • Click Print. A prompt tells you your report is being prepared. Once it has been prepared it is displayed as a PDF and you can then print the PDF file.
Export accounts list?
  • Export a list of the accounts to a CSVThe name given to a file which stores values that can be read by computer programs. Each line in the file is a record and each record is separated by a comma. file by clicking Export CSV. The list of transactions is automatically exported to a report.csv file.
