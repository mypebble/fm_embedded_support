Update Bank Ledger Code

About the fields
Code

The ledger code which is used to group transactions for a particular purpose or the bank ledger code which is used to match the bank in Fund Manager to the bank code in an external financial system.

The code can be AlphanumericNumbers, letters or a mix of both., up to 20 characters. If you integrate with an external accounting package for reporting purposes, set up the codes in Fund Manager that match the same codes in the external system you are integrating with. This ensures that the transaction appears in the external ledger that you want it to, making it easy for you to identify and track the transactions or if a bank ledger code that the transactions are posted to the expected banks in both systems.

Name

The name of the code.

Note: The VAT and Cash in Hand ledger codes are set up in Fund Manager by default and you can’t amend the name of these codes.

In use

The In use flag only applies to ledger codes you have created yourself. It doesn’t apply to the VAT or Cash in Hand codes or to bank ledger codes.

The flag indicates whether the ledger code is in use or not: selected, the code is in use; clear, it is not in use. If a code is not in use then you won’t be able to select it for transactions.

Bank account

The name of the bank account, selectable from the drop-down list of available bank accounts.

Note: You can only select a bank account that has already been created using Administration > Manage Bank Accounts.

How do I ...

Amend a ledger code
  1. Amend the existing Code.
  2. Amend the existing Name.

    3. Note: You won’t be able to do this if you are amending the code for the default VAT or Cash in Hand ledger codes.

  3. Select or clear the In use flag.
  4. Click Save.
Amend a bank ledger code
  1. Amend the existing Code.
  2. Select a different Bank account.
  3. Click Save.
Create a ledger code
  • From Administration > Manage Ledger Codes, select New Ledger Code and follow the instructions on the Update Ledger Code screen.
Create a bank ledger code
  • From Administration > Manage Ledger Codes, select New Bank Ledger Code and follow the instructions on the Update Ledger Code screen.
Mark a ledger code in use

Marking a ledger code as in use means that you can use the ledger code and apply it to transactions.

Note: Marking a code in use or out of use does not apply to Bank Ledger codes.

  1. Select the In use flag.
  2. Click Save.
Mark a ledger code out of use

Marking a ledger code as out of use means that you can no longer use the ledger code and apply it to transactions.

Note: Marking a code in use or out of use does not apply to Bank Ledger codes.

  1. Clear the In use flag.
  2. Click Save.

    3. You won’t be able to use the ledger code
