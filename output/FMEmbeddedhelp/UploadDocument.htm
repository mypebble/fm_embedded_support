About the fields
Name

The name of the document that you want to upload. This should make it easy to identify which document you are looking at e.g. Y1 - 160830 Trip.

Description

The description of the document you are uploading.

Document

Choose a document to upload.

Tip: We recommend uploading PDF documents as these are easier for users to open and view.

Public

Selected means the document is shown to all Joinos users of the school.

Warning! Do not upload any private or sensitive information because marking a document as non-public does not stop users from downloading it.

Use this screen to upload documents you want to attach to offers, for example, trip itineraries or cost breakdowns, or newsletters.

Warning! Do not upload any private or sensitive information.

How do I ...

Upload a document?
  1. Enter a name for the document.
  2. Enter a description for the document.
  3. Choose the document by clicking Choose File and browsing to where it is on your computer network.
  4. Select Public if it should be visible to all members of the school.

    5. Note: Do not upload any private or sensitive information because marking a document as non-public does not stop users from downloading it.

Attach a document to an offer?
  1. Use Offers > Active Offers to find the offer you want to attach the document to.
  2. Click Amend to display the offer details.
  3. Click Add a file you uploaded previously.
