Who’s Not Paid

About the fields
Account

Select the user account.

Note: Accounts are only listed for selection if they have a unit cost that isn’t zero (0). Unit costs are assigned using Administration > Manage Accounts and represent the cost per head for a trip or activity.

Date From/To

Select the start and end date for the report if required.

Show Paid Balances

Select this optional check box to show in the report all who have paid in full, part-paid or not paid at all.

Year/Class/Group

Select the Year, Class or Group from the drop-down list which you want to report on.

Search

Click to search using your criteria for who hasn’t paid. The list of names found is displayed as a result of the search.

List of Names

The list of names between the start and end date for the report, in the year, class or group, selected before clicking Search.

The list displays the Name, Class, Year, Reg, Receipts, Payments, Balance and Outstanding balance.

Filter

Enter a character or characters you want to find and the list of displayed names adjusts accordingly to show a shorter list.

This report lets you see a list of who’s not paid in a year, class or group, between two dates. .

How do I ...

Use the search criteria?
  1. Select the Account.
  2. Enter the start and end date to display the results you want.
  3. Select Show Paid Balances if you want to include in the list those who have fully paid.
  4. Select the Year, Class or Group and choose from the drop-down list.
  5. Click Search.
Print list?
  • Click Print. A prompt tells you your report is being prepared. Once it has been prepared it is displayed as a PDF and you can then print the PDF file.
Send a text message to ask for payment?
  • Click Send SMS.

    • An attempt is made to send a text message requesting payment to all on the list.

    Tip: If there is no phone number to send an SMS to, you are prompted to manage the phone numbers for the names on the list. However this is done after an attempt to send the messages. Therefore, you can check the names on the list before you try to send the messages to ensure they have phone numbers. This is done through Administration > Manage Names > Manage Phone Numbers.

Export list?
  • Export a list of the names to a CSVThe name given to a file which stores values that can be read by computer programs. Each line in the file is a record and each record is separated by a comma. file by clicking Export CSV, choosing the location and name to save the CSV file and then clicking Save.
